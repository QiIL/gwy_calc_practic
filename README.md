# gwy_calc_practic

一个纯静态的公务员速算练习页，直接打开 [`index.html`](/root/gwy_calc_practic/index.html) 就可以使用。

## 当前结构

- 配置页：选择考试模式、题目数量、计时模式、答案显示策略
- 考试页：顺序计时或倒计时答题，支持快速结束
- 总结页：展示统计结果、错题回看、最近成绩记录

## 这次新增

- 从单页题目列表改成了三段式考试流程
- 首页改成卡片式配置布局
- 支持顺序计时和倒计时
- 支持即时反馈或总结页统一看答案
- 总结页新增错题回看
- 总结页新增浏览器本地成绩记录

## 说明

- 历史成绩保存在浏览器 `localStorage`
- 项目不依赖构建工具，适合直接部署到任意静态站点
